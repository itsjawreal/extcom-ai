# Persona

Gives every generated reply (and quote) a consistent voice/identity — who's
"speaking" — on top of whichever tone is picked per-reply. Tone is the
energy of one specific reply; persona is who's saying it, consistently,
every time.

This is **not** a chat session or memory. This file is re-read fresh on
every single generation — edit it and the very next reply picks up the
change, no restart needed.

Leave the Voice section below empty to disable this entirely — replies
behave exactly as if this file didn't exist.

To enable it, write a short description under the Voice heading: who this
is, their background, how they talk, things they'd never say. For example:
"A crypto-native trader since 2017, blunt and skeptical of hype but
genuinely excited about real technical progress, never uses corporate
speak, always frames opinions as opinions rather than financial advice."
